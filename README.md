## Conversor DBF para CSV

O reposiótio armazena o script que recebe um arquivo em DBF e retorna convertido em CSV, utilizando a classe simpledbf escrita em python

Dependências:
  - Pandas
     > - pip install pandas
  - Numpy
     > - pip install numpy
  - simpledbf
     > - pip install simpledbf
